## Introduction

Landslides are dangerous natural disasters that cause environmental and economic damage. Early detection can help reduce risk and improve disaster response planning.

In this project, I developed a deep learning model to detect landslides from satellite imagery using semantic segmentation.

---

## Problem

The main challenge was extreme class imbalance.

Only about 1.9% of pixels represented landslides, while the rest were background. Because of this, accuracy was not a reliable metric.

Instead, I focused on:

- F1 Score
- Recall
- Precision

---

## Model Architecture

I used:

- **UNet++** architecture  
- EfficientNet-b5 encoder  
- Binary segmentation output  
- Data augmentation  
- Loss function adjusted for imbalance  

---

## Results

The model achieved:

- Accuracy: ~0.98  
- Improved F1-score compared to baseline UNet  
- Strong recall performance for minority class
