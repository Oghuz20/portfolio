## Overview

This project implements a Convolutional Neural Network (CNN) to classify images as either cats or dogs.

The goal was to understand the fundamentals of computer vision and deep learning model training.

---

## Model Architecture

- Convolutional layers with ReLU activation
- MaxPooling layers
- Fully connected dense layers
- Sigmoid output for binary classification

---

## Training Process

- Image resizing and normalization
- Data splitting (train / validation)
- Binary cross-entropy loss
- Accuracy metric evaluation

---

## Results

- High training accuracy
- Good generalization on validation data
- Proper classification of unseen images

---

## Technologies Used

- Python
- TensorFlow / Keras
- OpenCV
- NumPy
