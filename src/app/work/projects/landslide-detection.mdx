## Overview

This project focuses on detecting landslides from satellite imagery using deep learning-based semantic segmentation.

Due to extreme class imbalance (landslide pixels ≈ 1.9%), the primary evaluation metrics were F1-score and Recall instead of overall Accuracy.

---

## Model Architecture

- UNet++ with EfficientNet-b5 encoder
- Binary segmentation (landslide vs non-landslide)
- Data preprocessing and augmentation applied
- Loss function adjusted for class imbalance

---

## Results

- Accuracy: 0.98+
- High Recall for minority landslide class
- Improved F1-score compared to baseline UNet
